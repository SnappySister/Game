# 双人回合制卡牌对战

一个基于 Node.js + WebSocket + 原生 HTML/JS 的局域网实时对战卡牌游戏，支持两人在线对决。

## 功能特点

- **实时对战**：基于 WebSocket 实现低延迟同步，两人在线即可开局
- **水晶费用体系**：类似炉石传说的水晶消耗机制，低费牌概率更高
- **后手补偿**：后手玩家开局赠送一张【水晶硬币】（0费，本回合水晶+1）
- **15 张独特卡牌**：覆盖伤害、治疗、护盾、吸血、中毒、格挡、偷牌、净化、自爆等效果
- **免疫格挡机制**：格挡后可完全免疫一回合所有伤害，连护盾都不消耗
- **真实伤害**：部分卡牌（如献祭、自爆卡车）直接扣减 HP，但免疫状态可免除
- **弃牌机制**：非硬币手牌数不能超过当前水晶上限，硬币不计入上限
- **再来一局**：对局结束后双方刷新页面即可重开，无需重启服务器

## 快速开始

```bash
cd card-game
npm install
node server.js
```

浏览器打开 `http://服务器IP:3456`，两人连接后自动开局。

## 游戏规则

1. **胜利目标**：将对手 HP（初始 80）降至 0。
2. **水晶**：每回合上限 +1（最高 10），回合开始时水晶回满。
3. **手牌上限**：非硬币手牌数不能超过当前水晶数，硬币不计入。
4. **抽牌**：开局各抽 4 张，之后每回合抽 3 张。
5. **伤害结算**：先扣护盾，再扣 HP。免疫状态下所有伤害（包括真实伤害）均无效。
6. **结束回合**：可随时结束让对手行动。若手牌超限，必须先弃牌。

## 部署到公网服务器

```bash
# 服务器安装 Node.js
apt update && apt install -y nodejs npm

# 上传项目后安装依赖
cd card-game
npm install

# 用 pm2 后台运行
npm install -g pm2
pm2 start server.js --name card-game
pm2 save
```

别忘了在云服务器安全组里放行 **TCP 3456** 端口。

## 项目结构

## 技术栈

- **后端**：Node.js + `ws` 包
- **通信**：WebSocket 原生协议
- **前端**：原生 HTML5 + CSS3 + JavaScript（无框架）
