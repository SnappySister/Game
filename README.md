# Game

一个支持本地部署、局域网多人联机实时对战的小游戏平台。

> **注意**：`card-game/` 目录已停止更新，当前所有开发维护都集中在 `lobby/` 目录。

## 快速开始

```bash
cd lobby
npm install
node server.js
```

浏览器访问 `http://localhost:3456`，输入昵称即可进入大厅。

同一局域网内的其他玩家可通过你的局域网 IP + `:3456` 加入对战。例如在 Windows 防火墙放行 3456 端口后，其他玩家访问 `http://192.168.1.x:3456`。详见 [`lobby/README.md`](lobby/README.md)。

## 项目简介

**lobby** 是一个基于 Node.js + WebSocket + 原生 HTML/JS 构建的局域网多人游戏大厅。采用模块化引擎架构，每个游戏以独立模块接入，未来可不断扩展新游戏。

## 当前游戏

| 游戏 | 人数 | 简介 |
|------|------|------|
| 卡牌对战 | 2 人 | 回合制策略卡牌，水晶与负面效果机制 |
| 德州扑克 | 2-6 人 | 无限注德州扑克，筹码跨局保留，支持边池与投票继续 |

## 特性

- 无需注册，昵称即登
- 大厅 + 房间两级结构，支持创建、加入、观战
- 双频道聊天（大厅频道 / 房间频道），未读消息红点提醒
- 房主控制对局开始与强制结束
- 引擎模块化，新增游戏只需在 `games/` 下实现标准接口并注册
